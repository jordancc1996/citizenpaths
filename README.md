## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/citizenpaths.git
cd citizenpaths

# Install dependencies
npm install
# or
pnpm install

# Start development server
npm run dev
# or
pnpm run dev

# Build for production
npm run build
# or
pnpm run build
```

## 🚀 Deployment Options

### Vercel (Recommended)
1. Push to GitHub
2. Connect repository to Vercel
3. Deploy automatically with `vercel.json` configuration

### Netlify
1. Push to GitHub
2. Connect repository to Netlify
3. Deploy automatically with `netlify.toml` configuration

### GitHub Pages
1. Enable GitHub Pages in repository settings
2. GitHub Actions will automatically deploy on push to main
